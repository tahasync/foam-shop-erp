Design notes: Search bar uses a focused teal border + soft glow ring when active, with a live-filtered dropdown showing matched text in bold/tinted (highlight) plus a compact "+" add button per result, and recent-item chips below for quick re-add. The Save Sale success modal replaces any raw text/debug output with a bottom-sheet showing a checkmark, itemized recap, and two clear actions. The Update dialog is a center modal (not bottom sheet, since it's an unprompted system-level interrupt) showing current → new version and a short changelog, with "Update Now" opening the GitHub release page and "Remind me later" dismissing without blocking the app.
