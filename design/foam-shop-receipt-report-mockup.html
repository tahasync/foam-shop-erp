Design notes: Receipt uses a branded teal header block (shop icon + name + tagline) instead of plain black text, a proper itemized table with columns instead of stacked lines, a tinted totals card ending in an emphasized grand-total row, a color-coded status badge (green "Fully Paid" or red "Balance Due" instead of just red text), and an optional QR code + thank-you footer. The Export Report reuses the same header/brand block, adds a Dashboard-style color-coded summary card grid at the top (so the report reads consistently with the in-app Dashboard), a properly styled data table with a solid teal header row and alternating row shading, and a footer with page numbers. The same visual language (colors, table styling, header block) should extend to the Excel export: bold header row with teal fill and white text, currency-formatted number columns, frozen header row, and a Summary sheet mirroring the top card grid before the detailed data sheets.
